SMP Plugins
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
The Imagemaps plugin allows you to place any image into the game. To use it:
1. Submit your image for approval in the ⁠smp-imagemaps channel (we respond quickly)
2. Once approved, use the /imagemaps place command on the server:
/imagemap place <filename> [frameInvisible] [frameFixed] [frameGlowing] [size]
<filename> is the name you provided in your submission.
[frameInvisible] can either be true or false. If true, the item frame that contains the maps will be invisible, making the image flush with the wall.
[frameFixed] can either be true or false. If true, the image cannot be destroyed by hitting it or removing the wall behind it.
[frameGlowing] can either be true or false. If true, the item frames containing the map are glowing item frames that are light.[size] specifies the width and height of the image in blocks, written as [width]x[height].
For example, /imagemaps place bedrock false true true 4x3. Will create an image that has the name “bedrock”, where the item frame IS NOT invisible, the frame IS fixed, the frame IS glowing, and has a size of 4 blocks wide by 3 blocks tall.
[frameInvisible] and [frameFixed] can be modified after the image has been placed. Right click the item frame using a wooden hoe to toggle [frameInvisible], and shift+right click the item frame to toggle [frameFixed].
The resolution of a 1x1 map is 128x128 pixels. To ensure your images appear high quality, consider scaling them to multiples of 128 before submission. For example, a 4x3 image would be 512x384 pixels.

With the TTV chat plugin, you can use Twitch emotes in the Minecraft in-game chat. They function just like they do on Twitch: just type the emote name. The full list of emotes can be found at https://github.com/antoKeinanen/ttvChatPlugin/wiki/Emotes, or you can use the /listemotes command in game.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
The Gsit plugin allows you to use /sit, /lay, /crawl, /spin, and /bellyflop. You can also right click on stairs, slabs, or carpets to sit on them. Your hand must be empty for this to work. In addition, you can right click players with an empty hand to ride them.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
With the Graves plugin, you have 3 hours to collect your items after you die. The items are stored in a gravestone that functions as a chest. When you mine the gravestone, you are given 90% of the experience points that you lost when you died. When you respawn after a death, you are given a compass that points to the location of your grave. Right clicking the compass gives you the number of blocks to your grave.
The /grave command provides additional information about your graves.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
With the Simple Voice Chat plugin, you can use a built-in voice chat in Minecraft. This plugin supports proximity chat, meaning the farther you get away from other players, the quieter your voice becomes. It also has a Groups feature which is similar to a Discord voice chat, allowing you to talk with players from any distance in the game.
To use this plugin, you must install the Simple Voice Chat mod and place it in your client-side mods folder.
This plugin only works for Java Edition players. To talk with Bedrock Edition players, you must use Discord.
Download the mod here: https://modrinth.com/plugin/simple-voice-chat/versions?l=spigot Once the mod is installed and you are on Minecraft:
Press V to access the main menu.
Press G to access the Groups menu.
More information can be found at https://www.spigotmc.org/resources/simple-voice-chat.93738/.

The AFKPLUS plugin allows you to enter an AFK mode. An "AFK" title appears beside your name in the player list to show others that you are AFK. You automatically enter AFK mode after 2 minutes. However, you can manually enter the mode using /afk. While AFK, you cannot be attacked by mobs. However, other players can still attack you.

To use this plugin, you must become a VIP by donating at least $5 to the Minecraft Club. More details can be found at ⁠help-support-us.

To set a custom leave message, type /setleavemessage <message>.

The Wiki Integration plugin allows you to read pages from the SMP Wiki website from within Minecraft. This is a useful tool for new players to become familiar with the history of the server. This plugin was created by @Ilya (Cat6172).
The wiki command is /wiki <topic>, where the topic can be any wiki page. This includes player names, locations on the server, and lore events on the server.
You can read the full wiki at https://uofa-minecraft-server.fandom.com/wiki/UofA_Minecraft_Club_Server_Wiki.

Mob spawners will drop when mined with Silk Touch.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Most mobs have a chance to drop their own head. The chance is inversely proportional to how common the mob is (that is, rare mobs will usually drop their head, whereas easily farmable mobs have very low head drop rates). Looting does increase drop rates. Zombies, Skeletons, and Creepers are unaffected by this (you still need a charged creeper to get their head). A list of the heads and their drop rates can be found at
https://web.archive.org/web/20230818184511/https://wiki.vanillatweaks.net/VanillaTweaks:More_Mob_Heads
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Players will drop their heads if killed by another player. Remember you need another player's permission to kill them. You can also get your own head by shooting yourself with a bow.

The DuraPing datapack will notify the player when a used item drops to 10% of its durability. You can customize how you are notified by using /trigger duraPing. No value arguments. This will send you a message in the chat with clickable buttons.

The warden will drop a Swift Sneak 3 book when killed by a player.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
The PlayerStats plugin allows you view and share game statistics.
To view a statistic, type /statistic <statistic> <me / player / server / top>. When typing the command, it shows the list of statistics to chose from.
The server option shows the total amount for everyone in the server, whereas top provides the top 10 players.
To share the statistics in chat, run the command for a statistic and click the [Share] button.

The Parties plugin allows you create private chat groups for factions.
To create a party, type  /party create <name>.To join a party, you have to be invited by an existing party member. To invite someone to a party, type  /party invite <username>. To kick someone from a party, type /party kick <username>.To leave a party, type  /party leave.
You can switch between the global chat and party chat by typing  /party chat. All of your messages will be sent in global chat or party chat, depending on which mode you are in.You can remain in global chat and send specific messages in party chat using /pc <message>. This is the same thing as /tell, except for parties.
You can set a description for your party by typing  /party desc <description>.You can view the info for party by typing  /party info <party>.You can list all online parties by typing /party list.

The Mount & Ride datapack allows you to mount certain mobs by throwing a saddle on them, and control them with a special rod. The rod can be purchased from a custom villager in the shopping district: ⁠#Shops

For those that feel like they procrastinate too much playing on our servers, and need some help logging off on time: The Self Control plugin allows you to tell the server to kick you at a given time with
/kickme at [time]
where the time is in h:mma format (ex. 1:34pm). This will happen on the next instance of the given time only (will not repeat the next day). You can also use
/banme at [time]
to be temporarily banned (for 1 hour) if you need extra help staying off the server, though please don't ask the moderators to unban you early if you do. If you want to use this on your own servers, the plugin can be found here https://github.com/Cat6172/SelfControl

To teleport to the Campus world, type  /mvtp campus

The complete list of commands for the multiverse plugin can be found at https://github.com/Multiverse/Multiverse-Core/wiki/Command-Reference. You do not have access to all of these commands.

The PlotSquared plugin allows you to claim build plots on the Freebuild world.
To claim the plot you are standing in, type /plot claim. You are the only person who can build in the plot that you've claimed, unless you specifically give other people permission to build in your plot. You can claim 2 plots. If you want to claim more, contact the Execs. If you want to merge plots, contact the Execs.
To give others permission to build in your plot when you are online, type /plot add <player>.
To give others permission to build in your plot when you are offline and use WorldEdit, type /plot trust <player>.
To remove these permissions for a player, type  /plot remove <player>.
To prevent a player from entering your plot, type /plot deny <player>.
‍To teleport to your own plot, type /plot home.
To teleport to someone else's plot, type /plot visit <player>.
The full list of commands can be found at https://intellectualsites.gitbook.io/plotsquared/commands. You do not have access to all of the commands.
You can also change a bunch of features in the plot, such as time of day and weather using flags commands: https://pika-network.net/threads/plotsquared-flags-guide.105126/
